# game
一个游戏社区，系统采用微服务架构设计，在分布式环境下利用SpringCloudAlibaba框架，通过业务划分，设计独立模块的微服务，拆分为社区服务、订单服务、购物车服务、支付服务、用户管理服务、商品管理服务、文件上传服务等模块，黑盒游戏社区平台风格是一个当下比较流行的游戏社区和第三方游戏购买服务平台，为用户提供社交服务和游戏购买服务的平台。
